# User API Project

A Flutter application built during the Hash Plus App Development Bootcamp.  
The project demonstrates fetching user data from a REST API and managing state using Provider.

##  Features
- Fetch users from a public REST API
- Display a list of users
- View detailed information for each user
- State management using Provider
- Clean separation of API, models, providers, and UI

##  Technologies Used
- Flutter
- Dart
- REST API
- Provider
- HTTP Package

##  API Used
https://jsonplaceholder.typicode.com/users

##  Project Structure
- `api/` – Handles API requests
- `models/` – Data models
- `provider/` – State management
- `screens/` – App screens
- `widgets/` – Reusable UI components

## Purpose
This project was developed as part of a training bootcamp to practice:
- API integration
- State management
- Clean Flutter architecture
